== Description ==
Text Messaging to a phone from your WordPress Admin panel is as easy as pie using TextToMe plugin by Tradesouthwest. Covers the top ten mobile carriers in the US. Enter cell phone number then select carrier and write the message. Does not get much easier than that.

Features
* Ten United States carriers included
* Validation for 10 digit only phone numbers to avoid long distance texting
* Will add carriers as I get them

== Installation ==
1. Upload texttome.zip to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for link to plugin from the "Settings" Menu

== Frequently Asked Questions ==
Q.: Can I add more carriers to the plugin?
A.: Yes, If you are good with PHP and HTML you may add into the drop down select form input and and all carrier email suffixes and they should work fine.
